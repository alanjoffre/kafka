# Instruções

# Instalação Docker e Docker-Compose
Baixe o Docker Desktop for Windows;
Faça a instalação;

Baixe a ultima versão WSL2 Linux Kernel updatepackage for 64 machines / Step 4
https://learn.microsoft.com/en-us/windows/wsl/install-manual

Abra o docker Desktop

Abra o PowerShell ou CMD

Baixe os arquivos para teste
https://gitlab.com/software-engineering-class/template

Acesse o diretório com os arquivos pelo PowerShell ou CMD

Digite o seguinte comando: docker-compose up --build

Abra o arquivo: docker-composer.yml pelo VSCODE, verifique a porta do: phpmyadmin (4001)
Acesse pelo browser: localhost:4001

# Instalação Apache Kafka

Basta acessar o diretório Kafka pelo PowerShell ou CMD
digite o seguinte comando: docker-compose up --build

Digite o seguinte comando: ./kafka-topics.sh --zookeeper 127.0.0.1:2181 --topic myTopic --create --partitions 3 --replication-factor 3
Digite o seguinte comando: ./kafka-topics.sh --zookeeper 127.0.0.1:2181 --topic myTopic --describe
Digite o seguinte comando: ./kafka-console-consumer.sh--bootstrap-server localhost:19092,localhost:29092,localhost:39092 --topic myTopic
Digite o seguinte comando: ./kafka-console-producer.sh--broker-list localhost:19092,localhost:29092,localhost:39092 --topic myTopic
Digite o seguinte comando: ./kafka-topics.sh--zookeeper 127.0.0.1:2181 --list
Digite o seguinte comando: ./kafka-topics.sh--zookeeper 127.0.0.1:2181 --topic myTopic --delete
